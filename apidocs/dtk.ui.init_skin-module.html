init_skin(project_name, project_version, skin_name, system_skin_dir, system_theme_dir)

source code 

Initialize skin easily.

Parameters:
  • project_name - Project name.
  • project_version - Project version.
  • skin_name - Default skin name.
  • system_skin_dir - Application's skin directory in system level, user space skin directory at ~/.config/project_name/skin .
  • system_skin_dir - Application's theme directory in system level, user space theme directory at ~/.config/project_name/theme .
Returns:
Return application theme. 
>>> from dtk.ui.init_skin import init_skin
>>> from dtk.ui.utils import get_parent_dir
>>> import os
>>> 
>>> app_theme = init_skin(
>>>     "deepin-ui-demo", 
>>>     "1.0",
>>>     "01",
>>>     os.path.join(get_parent_dir(__file__), "skin"),
>>>     os.path.join(get_parent_dir(__file__), "app_theme"),
>>>     )

Equal to below code:

>>> from dtk.ui.skin_config import skin_config
>>> from dtk.ui.theme import Theme, ui_theme
>>> from dtk.ui.utils import get_parent_dir
>>> import os
>>> 
>>> # Init skin config.
>>> skin_config.init_skin(
>>>     "01",
>>>     os.path.join(get_parent_dir(__file__), "skin"),
>>>     os.path.expanduser("~/.config/deepin-ui-demo/skin"),
>>>     os.path.expanduser("~/.config/deepin-ui-demo/skin_config.ini"),
>>>     "deepin-ui-demo",
>>>     "1.0"
>>>     )
>>> 
>>> # Create application theme.
>>> app_theme = Theme(
>>>     os.path.join(get_parent_dir(__file__), "app_theme"),
>>>     os.path.expanduser("~/.config/deepin-ui-demo/theme")
>>>     )
>>> 
>>> # Set theme.
>>> skin_config.load_themes(ui_theme, app_theme)
